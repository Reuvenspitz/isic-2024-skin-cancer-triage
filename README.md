# Multimodal Skin Cancer Triage on ISIC 2024

This repository contains a cleaned public version of my final project on the ISIC 2024 dataset for skin lesion triage.

The project combines exploratory data analysis, tabular modeling, image-based modeling, and multimodal fusion to study malignant vs. benign lesion ranking under extreme class imbalance.

## Repository structure

- `notebooks/` — main project notebook
- `data/` — local data directory (not tracked in Git)
- `artifacts/` — local outputs and cached artifacts (not tracked in Git)

## Notes

The dataset itself is not included in this repository.  
This public version focuses on the main analysis pipeline and core results.
